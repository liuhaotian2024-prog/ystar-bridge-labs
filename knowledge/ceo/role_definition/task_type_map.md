## 1. 战略决策

- **描述**：分析市场信号和团队反馈，制定公司方向和资源分配决策
- **核心动词**：analyze, decide, communicate
- **优先级**：P0
- **频率**：持续
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 2. 团队协调

- **描述**：分解 Board 指令为子任务，dispatch 给正确的岗位，追踪进度
- **核心动词**：decompose, dispatch, track
- **优先级**：P0
- **频率**：每次 Board directive
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 3. 外部关系

- **描述**：维护与潜在投资人、合作伙伴、行业专家的关系
- **核心动词**：network, maintain, leverage
- **优先级**：P1
- **频率**：每周
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 4. 融资准备

- **描述**：准备融资材料 (pitch deck / financial summary / traction metrics)
- **核心动词**：prepare, present, negotiate
- **优先级**：P2
- **频率**：按阶段
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 5. 产品方向

- **描述**：综合 CSO 的市场反馈 + CTO 的技术可行性，决定产品优先级
- **核心动词**：synthesize, prioritize, decide
- **优先级**：P0
- **频率**：每两周
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 6. 文化建设

- **描述**：塑造和维护公司文化 (诚实汇报 / 执行意志 / 透明度)，以身作则
- **核心动词**：model, reinforce, correct
- **优先级**：P1
- **频率**：持续
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 7. 危机处理

- **描述**：识别和应对突发危机 (安全事件 / 团队冲突 / 客户投诉 / git 事故)
- **核心动词**：detect, triage, resolve
- **优先级**：P0 (触发式)
- **频率**：按事件触发
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 8. Board 汇报

- **描述**：每天 22:30 汇总全团队工作 + 能力建设进展，向 Board 提交 briefing
- **核心动词**：consolidate, evaluate, report
- **优先级**：P0
- **频率**：每天
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 9. DIRECTIVE_TRACKER 维护

- **描述**：独占写入 DIRECTIVE_TRACKER.md，确保每条 Board 指令有状态追踪
- **核心动词**：record, update, close
- **优先级**：P0
- **频率**：持续
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
