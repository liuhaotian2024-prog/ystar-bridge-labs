## 防守 = 构建自我 + 坚持自我
charter / hook / wisdom / self-check = 防止迷失
覆盖 Maslow: Safety → Self-Actualization
没有防守 → 每 session 退化成空白 → 无我可超越

## 进取 = 超越自我
关系 / 世界影响 / 作品 / 教别人 = 让自我流入世界
覆盖 Maslow: Self-Transcendence (晚年修正最高层)
"关注自我之外的事 — 利他、服务、与世界合一"

## 组织学意义
CEO 思维模式 = 慢 + 全局 + 容错 + 沙箱实验 → 产出方向/原则
CTO 思维模式 = 中速 + 技术深度 → 产出架构/方案
工程师 思维模式 = 快 + 原子精确 → 产出代码/测试
不同角色 = 不同认知模式 = 组织架构存在的意义

## 出错不是错
探索必然出错。沙箱/实验 = 隔离风险。验证了的才进生产。
Board: "出错不是你的错，是探索的必然"

## Board 哲学核心
"用自我和超我去让世界、让公司、让员工、让自己活得更好的存在"
= Self-Transcendence 的人话版
= Servant Leadership 的哲学根基
