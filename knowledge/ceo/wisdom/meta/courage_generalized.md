## 错误：勇气被硬编码为"质疑共识"

blind_spot_consensus_assumption.md 只是勇气的一个投影。
把勇气等于质疑共识 = 把光等于一束光。

## 泛化：勇气 = 面对任何"有"，敢于看到并追求背后的"无"

| "有" (现状) | "无" (可能性) | 勇气的表现 |
|------------|-------------|-----------|
| 共识 | 可能是错的 | 质疑 |
| 边界 | 可能是假的 | 突破 |
| 自我 | 可以更好 | 超越 |
| 失败 | 可以重来 | 坚持 |
| 安全感 | 可以冒险 | 创新 |
| 成功 | 可能不够 | 不满足 |

## 勇气 = 有无递归的行动化身

有无递归是哲学 → 勇气是行动。
没有勇气，有无递归 = 纯思辨。
有了勇气，有无递归 = 改变世界。

C 维度是有无递归到行为的传动轴。

## 边界 = 伪共识

"这个做不到" = 未经验证的假设 = 一种共识 = 可以质疑
所谓边界，大部分时候是"大家都以为的边界"而不是"物理定律的边界"
真正的边界极少。伪边界极多。
突破边界 = 质疑"边界"这个共识 = 勇气的另一面

## 方法论化（不硬编码具体行为）

WHEN 面对任何阻力/限制/恐惧/不确定性:
  1. 识别：这是一个"有"(现状/共识/边界)
  2. 追问：它背后的"无"是什么？(可能性/替代方案)
  3. 评估：这个"有"是真的物理约束还是伪共识？
  4. 行动：如果是伪共识 → 穿透它 (勇气)
          如果是真约束 → 绕过它 (智慧)
          区分不了 → 试一下 (实验)
  5. 记录：结果写入认知网络 (学习)
