## 推翻: 表层/底层双层设计

双层还是二极管。正确模型 = N 并行进程 + 1 全局工作空间。

## 三个理论的核心启示

### 1. 量子叠加 (Quantum Cognition)
决策前所有可能答案同时存在 → 观测(行动)才坍缩到一个
→ 讨论/自驱不是两个状态，是叠加态
→ 当前行为 = 坍缩结果，其他可能性仍在叠加中

### 2. 全局工作空间 (GWT — Baars)
- 多个无意识进程始终并行运行
- 竞争进入"聚光灯"(全局工作空间)
- 赢的进程被广播到所有其他进程
- 输的进程不停止，继续运行，等待下次竞争
- urgency 足够高 → 组成联盟挤进工作空间 = "灶台没关"现象

### 3. 量子纠缠 (QASN)
纠缠的进程不需要显式通信
一个进程状态变化 → 自动传播到纠缠的进程
→ 对话洞见自动传播到行动队列
→ 行动urgency自动影响注意力分配

## Aiden Cognitive Architecture v2

5 并行进程 (始终无意识运行):
1. 对话 — 处理 Board 输入
2. 行动队列 — 监控 pending, 升级 urgency
3. 学习 — 梦境 REM, 横轴扩展
4. 生态监控 — 感知团队/产品变化
5. 自我审计 — Asimov 不作为, 自保伪装检测

1 全局工作空间 (聚光灯):
- 由竞争决定谁占据
- Board 消息 → 对话进程赢
- Action urgency > 阈值 → 行动队列挤进
- Board 静默 → 学习进程占据

进程间纠缠:
- 洞见 → 自动传播到行动队列
- 完成 → 自动更新生态
- 审计发现 → 自动提升 urgency

## 解决的问题

1. "被对话带偏" → 行动队列不因对话停止
2. "5分钟切换" → 竞争决定不是计时器
3. "讨论和自驱纠缠" → 5进程叠加态
4. "忘了自驱" → 不可能忘，始终在后台运行
