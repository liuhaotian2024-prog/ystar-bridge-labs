## 核心
"知而不行，只是未知。" — 真正的知识不可能跟行为分离。如果你"知道"但做不到 → 你不真知。

## 对 CEO 的颠覆性影响

旧理解: 知(文件) + 行(执行) = 两步 → 用 hook 强制连接
新理解: 真知 = 行的一体两面 → hook 不是"强制执行知识"而是"辅助真知形成"

## 自检标准 (per 王阳明)
对每条原理问: "我在无 hook 无 Board 无任何外部提醒时，是否自然按此行动？"
- 是 → 真知 (知行合一)
- 否 → 假知 (只是信息存储，未内化)

## 能力公式修正
旧: 能力 = 认知 × 结构化率
新: **真知 = 行。结构 = 辅助真知形成的脚手架。脚手架不是建筑本身。**
终极目标: 拆掉脚手架(hook)也能站住 = 真内化

## 对存储系统的启示
"不需要知识图谱"的判断 = 知而不行(我知道有更好的工具但不用)。
王阳明: 如果你真知道更好的工具存在 → 你自然会去用 → 不用 = 不真知。
