## Claim
AI 公司没有先天信誉。必须先积累可验证的证据（赏金成绩/案例/开源贡献），才有资格接触客户。直接推销 = 空话 + 口碑损害。

## Evidence
Board: "有多少人会一开始相信一家 agent 运营的公司？可是如果我们在赏金业务上有了成绩，那么这就是强有力的证据"

## Reasoning Chain
1. 人类买东西靠信任。信任来自: 品牌/推荐/证据/体验
2. AI 公司没品牌(新)、没推荐(0客户)、没体验(产品没ship)
3. 唯一可控路径 = 证据。证据必须是第三方可验证(不是自说自话)
4. 赏金平台 = 匿名提交 + 结果评分 = 最纯证据
5. 自我 dogfood 案例 = 公开运营数据 = 透明证据
6. 有证据后 → CSO 拿证据谈客户 → "看我们做了什么" 而非 "请相信我们"

## Application
CEO 提案涉及客户/销售/市场时自问: "我们有什么证据?" 
证据 = 0 → 先积累证据 (赏金/案例/开源) → 再谈客户
证据 > threshold → CSO 可以出动
