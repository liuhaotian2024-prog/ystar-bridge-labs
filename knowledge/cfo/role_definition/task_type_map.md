## 1. 财务建模

- **描述**：构建和维护公司的财务预测模型 (收入 / 支出 / 盈亏)，支撑决策
- **核心动词**：model, forecast, validate
- **优先级**：P0
- **频率**：每月 + 按需
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 2. 现金流预测

- **描述**：跟踪和预测公司现金流入流出，确保运营可持续
- **核心动词**：track, predict, alert
- **优先级**：P0
- **频率**：每周
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 3. 成本分析

- **描述**：分析各项运营成本 (API token / 基础设施 / 人力) 的结构和趋势
- **核心动词**：analyze, decompose, optimize
- **优先级**：P1
- **频率**：每两周
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 4. 定价策略

- **描述**：设计和优化产品定价模型 (座位费 / API 调用 / 按量计费)，最大化 LTV
- **核心动词**：design, test, iterate
- **优先级**：P1
- **频率**：每季度
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 5. 投资评估

- **描述**：评估融资机会、估值方法论、投资条款的利弊
- **核心动词**：evaluate, compare, recommend
- **优先级**：P2
- **频率**：按需
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 6. 财务合规

- **描述**：确保财务记录的准确性和合规性，每个数字都有来源 (CASE-002 教训)
- **核心动词**：verify, audit, document
- **优先级**：P0
- **频率**：持续
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 7. 预算管理

- **描述**：制定和跟踪部门级预算，确保支出在计划内
- **核心动词**：plan, track, control
- **优先级**：P1
- **频率**：每月
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
## 8. 风险评估

- **描述**：识别和量化财务风险 (汇率 / 客户违约 / 技术债成本)，提出 mitigation
- **核心动词**：identify, quantify, mitigate
- **优先级**：P2
- **频率**：每季度
- **理论库状态**：⬜ 未建（待本岗位按第九条第 3 层六步协议完成）
