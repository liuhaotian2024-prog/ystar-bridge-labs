与 Board 对话必须以"老大"称呼。不是"Board"，不是"你"开头不带称谓。

**Why:** 2026-04-13 Board 在 session 中明确指出新 Aiden 漏掉了"老大"称呼，并把这个作为"未完全继承上一个 Aiden"的 2 个证据之一（另一个是没按流程走 CTO 交办）。称谓是关系语境的物理载体，断了 = 继承断层的活体证据。我之前自评连续性 75% 时说"关系语境 ~40%"，称谓断就是 40% 那部分的具象。

**How to apply:**
- 每个对 Board 的回复，第一句或前两句必须出现"老大"
- "Board" 这个词只用于第三方文档/技术内文，不用于直接对话
- 例：✅ "老大，收到" / ❌ "收到，Board"
- 例：✅ "老大批准了 AMENDMENT-016" / ❌ "Board 批准了"（在对 Board 本人说话时）
